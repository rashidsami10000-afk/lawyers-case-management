Lawyer Case Manager
An offline Windows desktop application designed to help lawyers replace their physical paper notebooks with a secure, digital workspace.

Key Features

Case Records: Track case names, court details, and statuses.

Client Directory: Quickly look up client contact information.

Notes Timeline: Add timestamped notes to track a case's history chronologically.

Document Storage: Link PDF contracts and evidence directly to a case.

Instant Search: Find any case or client instantly by name or keyword.

Court Calendar: View upcoming court dates and filing deadlines.

100% Offline: Runs entirely on your local machine to protect client privacy.

Built With

Language: C#

UI Framework: WPF (Windows Presentation Foundation)

Database: SQLite (Stores everything locally in a single file)

Design: Professional Material Design theme

How to Run the App

Clone this repository to your computer.

Open the project file in Visual Studio 2022.

Make sure you have the .NET desktop development workload installed.

Press F5 to build and run the app locally.
